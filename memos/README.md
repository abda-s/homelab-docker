# Memos Service

This directory contains the Docker configuration for **Memos**, a privacy-first, lightweight note-taking service.

## Overview

Memos provides a clean interface to capture thoughts and ideas. It is configured to store data in your home directory (`~/.memos`).

## Access

*   **URL**: `http://memos.elwahsh.home`
*   **Port**: `5230`

## Structure

*   `docker-compose.yml`: Defines the Memos service.

## Usage

To start the service:

```bash
docker compose up -d
```
